# PF Cashflow - 프로젝트 파이낸싱 캐시플로우 관리 시스템

## 프로젝트 개요
현대엔지니어링 주택영업1부 PF 사업용 웹 기반 SaaS 플랫폼

### 목표
- Excel 기반 PF 캐시플로우 모델의 디지털화
- 수익흐름, 건설비용, 금융구조 시각화
- 시공사, 시행사, 수탁사, 증권사용 통합 솔루션

### 기술 스택
- Frontend: Next.js
- Backend: Python
- UI/UX: Uizard
- AI Support: Claude, GPT-4, Cursor

## 개발 현황
- 단계: MVP 개발 (Solo)
- 계획: 확장성 고려한 설계
